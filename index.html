🔥 Firebase Setup

Hapat për sinkronizim real (falas):
1. Shko te console.firebase.google.com
2. Krijo projekt → Realtime Database → Test mode
3. Aktivizo edhe Storage (për fotot) → Test mode
4. Project Settings → Web App → Kopjo config
5. Zëvendëso firebaseConfig në kodin HTML
6. Ngarko në Netlify.com/drop — gati!

Pse foto pa limit tani: fotot ngarkohen te Firebase Storage, jo te databaza. Databaza ruan vetëm linkun e fotos, kështu që mund të shtosh sa foto të duash pa e ngadalësuar apo bllokuar sistemin.
